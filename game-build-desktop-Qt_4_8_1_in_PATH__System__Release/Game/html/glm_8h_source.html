00170 /* glmVertexNormals: Generates smooth vertex normals for a model.
00171  * First builds a list of all the triangles each vertex is in.  Then
00172  * loops through each vertex in the the list averaging all the facet
00173  * normals of the triangles each vertex is in.  Finally, vectors the
00174  * normal index in the triangle for the vertex to the generated smooth
00175  * normal.  If the dot product of a facet normal and the facet normal
00176  * associated with the first triangle in the list of triangles the
00177  * current vertex is in is greater than the cosine of the angle
00178  * parameter to the function, that facet normal is not added into the
00179  * average normal calculation and the corresponding vertex is given
00180  * the facet normal.  This tends to preserve hard edges.  The angle to
00181  * use depends on the model, but 90 degrees is usually a good start.
00182  *
00183  * model - initialized GLMmodel structure
00184  * angle - maximum angle (in degrees) to smooth across
00185  */
00186 
00187 GLvoid
00188 glmVertexNormals(GLMmodel* model, GLfloat angle);
